# Auditory Perception Simulator

A human-in-the-loop experiment for estimating auditory frequency discrimination thresholds using adaptive control algorithms.

---

## Overview

This program tests how well participants can perceive differences in pitch. Users listen to pairs of tones and indicate which one is higher. The system adapts the difficulty to estimate their auditory threshold.

- Interactive GUI
- Adaptive staircase algorithm
- Automatic data saving
- Performance visualization

---

## Running the Experiment

1. Download the `.exe` from the Releases page  
2. Double-click to launch  
3. Select or enter a participant ID  
4. Click **Start** and complete the trials  

---

## Output

- CSV files saved in the `data/` folder  
- Session performance plots saved in `reports/`  
- Accuracy and threshold estimates displayed after completion
